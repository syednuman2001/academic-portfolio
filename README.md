# Academic Portfolio Website

A modern academic portfolio website built using React.js and Vite for showcasing research, projects, publications, technical skills, and academic experience.

Features:
- Responsive modern UI
- Dark / Light mode
- Research portfolio showcase
- Publications section
- Projects section
- Technical blog
- Dynamic JSON-based content management
- Contact form with EmailJS
- Embedded CV page
- Mobile responsive navbar
- Academic timeline sections
- GitHub & social integration

Tech Stack:
- React.js
- Vite
- CSS3
- React Router
- React Icons
- EmailJS

Research Focus:
- Computer Vision
- Few-Shot Learning
- Vision Transformers (DeiT)
- Explainable AI
- Incremental Learning
- Domain Adaptation
- Okra Leaf Disease Detection using DeiT + Mistral-7B

Installation:
npm install
npm run dev

Deployment:
Deployed using Vercel.
